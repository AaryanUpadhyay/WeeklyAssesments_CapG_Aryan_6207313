# CG Assessment

This repository contains weekly Playwright automation assessment tasks.

## Project Structure

- `Assesment_week2_28-02-2026` - basic Playwright test cases and screenshots
- `Assesment_week3_07-03-2026` - task-based Playwright tests
- `Assesment_week4_14-03-2026` - file handling, test data, and upload/download related tasks
- `Assesment_week5_20-03-2026/open_cart_FW_Scenerio1` - OpenCart automation framework using Page Object Model and JSON test data

## Tech Stack

- TypeScript
- Playwright
- Node.js

## Setup

```bash
npm install
npx playwright install
```

Run these commands inside the specific weekly folder you want to execute.

## Run Tests

```bash
npx playwright test
```

For the OpenCart project:

```bash
cd Assesment_week5_20-03-2026/open_cart_FW_Scenerio1
npx playwright test
```

## Notes

- Each week contains separate assessment tasks.
- Screenshots and test data files are included in the respective folders.
- Week 5 follows the Page Object Model structure.
